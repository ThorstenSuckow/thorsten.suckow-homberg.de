# Stereotype (UML)

A **Stereotype** in **UML** is a formal extensions for available model elements in **UML**. It allows for providing
additional semantic context or to specify the semantic context in an otherwise ambiguous situation.

Although **UML** has a wide range of predefined **Stereotypes** that can be used to classify model elements, it also
allows for defining custom **Stereotypes**.

:::info Notation
The correct encapsulation of an identifier for a **Stereotype** is by using an opening and a closing [guillemet](https://en.wikipedia.org/wiki/Guillemet)
(`U+00AB LEFT-POINTING DOUBLE ANGLE QUOTATION MARK, “«”`, `U+00BB RIGHT-POINTING DOUBLE ANGLE QUOTATION MARK, “»”`),
often confused with two subsequent `U+003C LESS-THAN SIGN, “<”` (`<<`) and `U+003E GREATER-THAN SIGN, “>”` (`>>`)

**wrong**: `<<create>>`

**correct**: `«create»`
:::
