# Spanning Tree (Graph Theory)

If $G$ is a _connected graph_, and $V(G)$ is the set of its _vertices_, then the **Spanning Tree** $T \sube G$  is a _minimal connected_ graph with $V(T) = V(G)$.

The edges $E(T) \setminus E(G)$ are called the _chords_ of $T$ in $G$.
